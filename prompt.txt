1. General Guidelines & Technical Stack

Use vanilla HTML / CSS / JavaScript (no React) or a framework like Next.js without React parts or a simple static site generator, but avoid React complexity.

Use GSAP (GreenSock Animation Platform) and ScrollTrigger / ScrollSmoother plugins to handle scroll animations, smooth transitions, pinning, etc.

Use SVGs / vector animations / Lottie where possible for scalable and crisp visuals.

Prioritize performance: lazy load images, minimize layout shifts, and keep animation CPU usage moderate.

Use CSS variables (custom properties) for theme colors, spacing, typography to make styling consistent.

Responsive design: all animations and layouts must work gracefully on mobile, tablet, desktop. Use media queries and conditional scroll triggers for small screens.

Accessibility: make sure keyboard navigation, focus states, alt text on images, sensible tab order, and no traps caused by animations.

Use ScrollSmoother for overall smooth scrolling and parallax effects. 
GSAP

Use ScrollTrigger for scroll-based animations like pinning, scrubbing, transitions. 
GSAP

2. Global Animation / Scroll Behavior

Wrap your content like this:

<body>
  <div id="smooth-wrapper">
    <div id="smooth-content">
      <!-- all sections here -->
    </div>
  </div>
  <!-- fixed / overlay items like nav can be outside or pinned -->
</body>


Initialize:

gsap.registerPlugin(ScrollTrigger, ScrollSmoother);

ScrollSmoother.create({
  smooth: 1,
  effects: true,
  smoothTouch: 0.6,
  normalizeScroll: true,
});


Use data-speed or data-lag attributes on elements to create parallax or lag effects (e.g. data-speed="0.5") thanks to ScrollSmoother. 
GSAP

For sections where you want the border radius effect (edges rounding as user scrolls), use clip-path or inset() animations instead of animating border-radius (better performance) 
GSAP
+1

Use ScrollTrigger with pin: true and scrub: true to pin a section during scroll, create layered transitions (cards sliding over each other), and snap behavior. 
GSAP

3. Color / Style / Visual Theme Suggestions

Choose a modern palette (e.g. dark/navy + accent (teal, gold, or medical green) + white or light backgrounds).

Use soft shadows, rounded corners, smooth gradients.

Use micro-interactions: hover effects, subtle motion, SVG path drawing, line animations, and reveal transitions on scroll.

Use typography scale: e.g. H1, H2, H3 consistent sizes, line heights, responsive adjustments.

5. Example Animation / ScrollTrigger Pseudocode Snippets
// Border radius / clip-path effect on hero
gsap.timeline({
  scrollTrigger: {
    trigger: ".hero-section",
    start: "top top",
    end: "bottom top",
    scrub: true,
    pin: true,
  }
})
.fromTo(".hero-wrapper", {
  clipPath: "inset(0% 0% 0% 0%)"
}, {
  clipPath: "inset(10% 10% 10% 10%)",
  ease: "none"
});

// Parallax speed effect using ScrollSmoother
// if element has data-speed attribute
// e.g. <div class="layer" data-speed="0.5">...</div>

// Pin & slide cards
let panels = gsap.utils.toArray(".project-panel");
panels.forEach((panel, i) => {
  ScrollTrigger.create({
    trigger: panel,
    start: "top top",
    end: () => "+=" + window.innerHeight,
    pin: true,
    pinSpacing: false,
    scrub: true
  });
});

// Reveal founder photo into mockup
gsap.timeline({
  scrollTrigger: {
    trigger: ".founder-section",
    start: "top 80%",
  }
})
.from(".founder-mockup svg", { scale: 0.8, autoAlpha: 0 })
.from(".founder-text", { x: 50, autoAlpha: 0, duration: 0.6 });
